### What is RevivalMobile?
*RevivalMobile is an experimental custom server bundle and app patches for various General Mobile and other old apps such as:*
- GM Puan
- GM Saha
- GM Sinema / Casper Sinema
- GM Asistan *(mainly Live Device)*
